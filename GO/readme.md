## Installatie
`Installeer Go`
`Installeer NPM / Node.JS`
`npm install touch-cli -g`

## Running
# Invoke Reverse
`go run main.go reverse foo`
> oof

`go run main.go rev bar`
> rab

# Invoke Inspect
`go run main.go inspect lorem`
> 'lorem' has 5 chars

`go run main.go insp FooBar`
> 'FooBar' has 6 chars
